Welcome to reliability-tool's documentation!
============================================

Welcome to the reliability-tool documentation. reliability-tool attempts to overcome the limitations of Language-Models on robustness.

This tool is developed in such a way that you don't need to do any coding as far as your pre-trained model is available on huggingface space or developed using transformers library.

If you are new to the reliability-tool, please use following links for installation guidance and example tutorial:

   :ref:`installation <installation>`

   :ref:`reliability-tool 101 <tool101>`

If you have already setup the tool, then feel free to refer following pages for BYoM (Bring Your own Model) based evaluations:

   :ref:`BYoM (Bring Your own Model) <byom>`

If you are a developer and interested in contributing then we highly suggests to refer following pages:

   Learn about the dependencies
   Learn the infrastructure
   Start contributing
